Welcome To DuxTree!

A low-resource and easy-to-use React tree component.

DuxTree is a React tree component with built-in support for lazy loading of tree nodes and optional checkable folders and nodes. Nodes can use custom rendering and the tree contents can be altered at runtime. All tree styles can be overridden in CSS.

Why Another Tree Component?

DuxTree offers the following notable features.

  • Low-resource and easy to use
  • Tree can be loaded dynamically with callbacks
  • Nodes and folders can use checkboxes
  • All nodes can be custom rendered with a callback
