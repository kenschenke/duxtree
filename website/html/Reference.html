Reference

Below is a list of every <DuxTree> property as well as a list of the array properties for the data property.

Property Default Value Description
data (none) The contents of the tree. See Populating a Tree for more information. A reference for the array is at the end of this page.
checkable false If true, checkboxes are shown with every node in the tree.
checkboxUnchecked (none) This can be used to customize the appearance of checkboxes in the unchecked state. See the page on Customizing Appearance for more information.
checkboxChecked (none) This can be used to customize the appearance of checkboxes in the checked state. See the page on Customizing Appearance for more information.
checkboxIndeterminate (none) This can be used to customize the appearance of checkboxes in the indeterminate state. See the page on Customizing Appearance for more information.
disclosureExpand (plus sign) This defines an optional React component to render to expand a folder. See the page on Customizing Appearance for more information.
disclosureCollapse (minus sign) This defines an optional React component to render to collapse a folder. See the page on Customizing Appearance for more information.
onExpand (none) This is an optional callback that is called whenever a user expands any folder in the tree. See the page on Callbacks for more information.
onCollapse (none) This is an optional callback that is called whenever a user collapses any folder in the tree. See the page on Callbacks for more information.
onSelectionChanged (none) This is an optional callback that is called whenever a user checks or unchecks any node in the tree. See the page on Callbacks for more information.

Tree Node Properties

Tree data is defined as JavaScript arrays. See the page on Populating a Tree for more information. Below are all possible values for each node in the tree.

Property Default Value Description
id (none) This property uniquely identifies each node in the tree. It must be unique across all nodes.
label (none) This is the label shown to the user. It can be plain text, a React component, or a function. If a function, the node's id is passed to the function which can return plain text or a React component. See the page on Customizing Appearance for more information.
icon (none) This defines an optional icon for this node. This is a React component to render. See the page on Customizing Appearance for more information.
show true If true, the node is rendered in the tree. This can be a boolean value or a function. The function is passed the ID of the node and must return true or false.
children (none) If present, this is an array of children for this node.
defaultExpanded false If true, the node's children are shown when the it is first rendered.
onExpand (none) This is called when a folder is expanded by the user. See the page on Callbacks for more information.
onCollapse (none) This is called when a folder is collapsed by the user. See the page on Callbacks for more information.
onLoadChildren (none) This is called when a folder is expanded by the user. See the page on Loading on Demand for more information.
