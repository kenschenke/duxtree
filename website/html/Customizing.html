Customizing Appearance

DuxTree's appearance can be controlled in a few different ways.

Labels

Item labels can be specified as a string, a JSX component, or a callback function. For a callback function, the node's ID is passed to the function. 

    const treeData = [
       {
          id: 'node1',
          // label is specified as a string
          label: 'Label for Node 1'
       },
       {
          id: 'node2',
          // label is specified as a JSX component
          label: <span className="red-label">Label for Node 2</span>
       },
       {
          id: 'node3',
          // label is specified as a callback function
          label: id => `Label for ${id}`
       }
    ];

Icons

An item can have a custom icon, specified with the icon property. 

    const treeData = [
       {
          id: 'folder1',
          label: 'Folder 1',
          icon: <i className="material-icons-outlined">folder</i>
       }
    ];

Disclosure Controls

You can specify the user controls to expand or collapse a folder. The default is a plus or minus symbol. The controls are specified as properties of the DuxTree component as disclosureExpand and disclosureCollapse. 

    <DuxTree
       ...
       disclosureExpand={<i className="material-icons-outlined">arrow_right</i>}
       disclosureCollapse={<i className="material-icons-outlined">arrow_drop_down</i>}
    />

Custom Checkboxes

You can specify user the controls for checkboxes. The default is HTML input tags. The controls are specified as properties of the DuxTree component as checkboxUnchecked, checkboxChecked, and checkboxIndeterminate. 

    <DuxTree
       ...
       checkboxUnchecked={<i className="material-icons">check_box_outline_blank</i>}
       checkboxChecked={<i className="material-icons-outlined">check_box</i>}
       checkboxIndeterminate={<i className="material-icons-outlined">indeterminate_check_box</i>}
    />

CSS

DuxTree is rendered using CSS classes. You can override the CSS to customize the tree appearance. Following is a list of the CSS classes.

.duxtree
The outer container for the entire tree.
.duxtree-node
The outer container for a tree node.
.duxtree-node-children
The container for a node's children.
.duxtree-disclosure
The container for a node's disclosure control to expand or collapse the children.
.duxtree-checkbox
The container for a node's checkbox.
.duxtree-icon
The container for a node's icon.
.duxtree-label
The container for a node's label.
.duxtree-loadingmsg
The container for the loading message used for dynamically loading children.
